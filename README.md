# 📖 Search Analytics <a name="about-project"></a>

**Search Analytics** is a rails application where users can search for multiple queries. It also provides analytics on the most searched items.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Ruby development environment

```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/amejid/search-analytics.git
```

### Install

Install this project with:

```sh
  cd search-analytics
  bundle install
  rails db:migrate
  rails db:seed
```

### Usage

To run the project, execute the following command:

```sh
  bin/dev
```

### Run tests

To run tests, run the following command:

```sh
  rspec
```
